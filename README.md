# 🚀 FMHY 中文版 (FreeMediaHeckYeah Chinese)

> **互联网自由媒介的中文百科全书** > 这是一个基于 [FMHY](https://github.com/fmhy/edit) 精神建立的中文资源导航。我们致力于收集最优质、纯净、无广告的中文互联网资源，并每日自动同步全球最新的技术与工具。

## 🌟 项目特色

* **每日同步：** 每天自动拉取原版 FMHY 全球更新，确保工具类资源与国际接轨。
* **本土化适配：** 针对中文用户习惯，深度整合网盘搜索、高清影视、汉化软件及学术资源。
* **纯净无碍：** 严格筛选掉带有欺诈广告、恶意捆绑的站点。
* **静态极速：** 基于 Docsify 驱动，无需数据库，全球 CDN 加速。

## 🛠️ 自动化原理

本项目采用 **GitHub Actions** + **Python** 实现全自动同步。
1. **Upstream：** 每天凌晨拉取 `fmhy/edit` 原始 Markdown。
2. **Injection：** 脚本自动将 `custom_*.txt` 中的中文特供资源注入页面顶部。
3. **Deployment：** 自动构建并推送到 GitHub Pages。
